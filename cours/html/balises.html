LES BALISES HTML

Nom de la balise Description
a Ancre HTML
abbr Abréviation
address Adresse
area Définit une zone particulière d'une image
article Représente une composition autonome dans une page (ex : un message d'un forum)
aside Représente une partie d'un document dont le contenu n'a qu'un rapport indirect avec le contenu principal du document (ex : un apparté)
audio Pour insérer un fichier audio
b Mettre en gras
base Définit l'URL de base à utiliser pour recomposer toutes les URL relatives contenues dans un document
bdi Isole une portée (span) de texte pouvant être formatée dans une direction différente de celle du texte qui l'entoure
bdo Utilisé afin d'outrepasser la direction du texte. Cela permet d'imposer une direction donnée à un texte.
blockquote Citation
body Corps du document
br Saut de ligne
button Bouton
canvas Permet de dessiner avec du javascript
caption Légende d'un tableau
cite Citation d'une oeuvre (livre, chanson,...)
code Représente un code informatique
col Définit une colonne appartenant à un tableau et est utilisé afin de définir la sémantique commune à toutes ses cellules
colgroup Définit un groupe de colonnes au sein d'un tableau.
data Relie un contenu à une version de ce contenu interprétable par un ordinateur.
datalist Contient un ensemble d'éléments option qui Représentent les valeurs possibles pour d'autres contrôles
dd Liste de définition (équivalent à un li)
del Représente une portion de texte ayant été supprimée d'un document
details Est utilisé comme un outil permettant de révéler une information. Un résumé ou un intitulé peuvent être fournis grâce à un élément summary
dfn Utilisé pour indiquer le terme défini dans le contexte d'une expression ou d'une phrase de définition
dialog Boite de dialogue
div Conteneur
dl Liste de définition (équivalent à un ul)
dd Liste de définition (équivalent à un li en gras)
em Est utilisé afin de marquer un texte sur lequel on veut insister
embed Permet d'intégrer du contenu externe à cet endroit dans le document. Le contenu peut être fourni par une application externe ou une autre source telle qu'un plugin du navigateur
fieldset Est utilisé afin de regrouper plusieurs contrôles interactifs ainsi que des étiquettes (label) dans un formulaire HTML
figcaption Représente une légende décrivant le reste du contenu de son élément parent figure
figure Représente un contenu autonome, éventuellement accompagné d'une légende facultative
footer Pied de page
form Formulaire
h1 Titre
head Inclure les feuilles de style, le titre,...
header Bloc d'entête (Il peut contenir des éléments de titre, mais aussi d'autres éléments tels qu'un logo, un formulaire de recherche)
hgroup Représente un titre de plusieurs niveaux pour une section d'un document. Il regroupe un ensemble d'éléments h1–h6.
hr Trait horizontal de séparation
html Représente la racine du document
i Mettre le texte en italic
iframe Permet d'intégrer une page HTML dans la page courante
img Permet d'insérer une image
inpt Est utilisé pour créer un contrôle interactif dans un formulaire web (ex : un champ de saisi)
ins Représente un fragment de texte qui a été ajouté dans un document.
kbd Représente une plage de texte en ligne indiquant la saisie de texte par l'utilisateur à partir d'un clavier, d'une saisie vocale ou de tout autre dispositif de saisie de texte
label Rattaché un élément de type input
legend Représente une légende pour le contenu de son élément parent fieldset
li Elémént d'une liste
link Définit la relation entre le document courant et une ressource externe (ex : un fichier CSS)
main Bloc qui représente le contenu majoritaire du body
map Est utilisé avec des éléments area afin de définir une image cliquable divisée en régions
mark Représente un texte marqué ou surligné à cause de sa pertinence dans le contexte.
menu Est une alternative sémantique à ul, mais est traité par les navigateurs, et en termes d'accessibilité comme un élément ul
meta Représente toute information de métadonnées qui ne peut pas être représentée par un des éléments (base, link, script, style ou title)
meter Représente une valeur scalaire dans un intervalle donné ou une valeur fractionnaire.
nav Section destinée à la navigation dans un document (avec des menus, des tables des matières, des index, etc.)
noscript Définit un fragment HTML qui doit être affiché si les fonctionnalités de script ne sont pas prises en charge ou si elles sont désactivées
object Représente une ressource externe qui peut être interprétée comme une image, un contexte de navigation imbriqué ou une ressource à traiter comme un plugin
ol Liste ordonnées
optiongroup Utilisé dans un formulaire, permet de créer un groupe d'options parmi lesquelles on peut choisir dans un élément select
option Elément d'un menu déroulant
outpout Représente un conteneur dans lequel un site ou une application peut injecter le résultat d'un calcul ou d'une action utilisateur.
p Paragraphe
picture Est un conteneur utilisé afin de définir zéro ou plusieurs éléments source destinés à un élément img. Le navigateur choisira la source la plus pertinente selon la disposition de la page (les contraintes qui s'appliquent à la boîte dans laquelle l'image devra être affichée), selon l'appareil utilisé (la densité de pixels de l'affichage par exemple avec les appareils hiDPI) et selon les formats pris en charge (ex. WebP pour les navigateurs Chromium ou PNG pour les autres).
pre Représente du texte préformaté
progress Indique l'état de complétion d'une tâche et est généralement représenté par une barre de progression
q Cet élément est destiné aux citations courtes qui ne nécessitent pas de sauts de paragraphe. Pour les plus grandes citations, on utilisera l'élément blockquote
rp Est utilisé pour fournir ce qui fera office de parenthèse aux navigateurs qui ne prennent pas en charge les annotations Ruby. Les annotations Ruby permettent d'afficher la prononciation des caractères d'Asie orientale, notamment lors de l'usage de caractères furigana Japonais ou bopomofo Taïwanais
rt Indique la composante texte d'une annotation Ruby, il est notamment utilisé pour la prononciation, la traduction ou la translitération des caractères d'Asie orientale. Cet élément est toujours contenu dans un élément ruby
ruby Sert à afficher la prononciation des caractères d'Asie orientale.
s Texte barré
samp Est un élément qui permet de représenter un résultat produit par un programme informatique en incise dans du texte
script Permet d'insérer du javascript
section Représente une section générique d'un document, par exemple un groupe de contenu thématique
select Meu déroulant
slot Représente un emplacement d'un composant web qu'on peut remplir avec son propre balisage
small Permet de représenter des commentaires ou des textes à écrire en petits caractères
source Définit différentes ressources média pour un élément >picture, audio ou video
span Mettre du texte
strong Indique que le texte a une importance particulière
style Permet d'insérer du CSS
sub Permet de mettre du texte en indice
summary Représente une boîte permettant de révéler le contenu d'un résumé ou d'une légende pour le contenu d'un élément details.
En cliquant sur l'élément summary, on passe de l'état affiché à l'état masqué (et vice versa) de l'élément details parent.
sup Permet du texte en exposant
table Tableau
tbody Permet de regrouper un ou plusieurs éléments tr afin de former le corps d'un tableau
td Cellule d'un tableau
template Mécanisme utilisé pour stocker du contenu HTML (côté client) qui ne doit pas être affiché lors du chargement de la page mais qui peut être instancié et affiché par la suite grâce à un script JavaScript
textarea Zone de saisie
tfoot Permet de définir un ensemble de lignes qui résument les colonnes d'un tableau
th Cellule d'entête d'un tableau
thead Définit un ensemble de lignes qui définit l'en-tête des colonnes d'un tableau
time Cet élément permet d'utiliser l'attribut datetime afin de traduire la date ou l'instant dans un format informatique (permettant aux moteurs de recherche d'exploiter ces données ou de créer des rappels)
title Titre de l'onglet
tr Ligne d'un tableau
track Est utilisé comme élément fils d'un élément audio ou video afin de générer des sous-titre
u Texte souligné
ul Liste non ordonnée
var Représente une variable dans une expression mathématique ou un texte lié à la programmation. Son contenu est généralement représenté avec une version italique de la police environnante utilisée
video Insérer une vidéo
wbr Permet de Représenter un emplacement où casser la ligne si nécessaire. Le navigateur pourra alors utiliser cet emplacement pour effectuer un saut de ligne si le texte est trop long et qu'en temps normal, une règle empêche le saut de ligne
